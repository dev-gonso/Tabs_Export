# Tabs Exporter and Opener

**Tabs Exporter and Opener** is a simple Firefox extension that lets you export all your open tabs to an HTML file and open all links from such an HTML file as new tabs.

## Features

- **Export Tabs:** Save all open tabs in your current Firefox window as clickable links in an HTML file.
- **Open Links:** Open all links from an HTML file (created by this add-on) as new tabs, in the same order as they appear in the file.
- Easy to use, no setup required.

## How to Use

1. **Install the Add-on:**  
   Add the extension to your Firefox browser.

2. **Open the Add-on:**  
   Click the add-on icon in your browser toolbar to open the popup window.

3. **Export Tabs:**  
   Click the **Export tabs** button.  
   An HTML file with all your open tabs will be downloaded to your computer.

4. **Open All Links:**  
   Open the previously exported HTML file in a Firefox tab.  
   Click the **Open all links** button in the add-on popup.  
   All links from the HTML file will open as new tabs, in order.

## Installation

You can install the extension from the [Firefox Add-ons Store](https://addons.mozilla.org/) (Tabs_Export)
